Método initObservable:

Este método cria um novo Observable:
  • O Observable é criado usando a classe Observable do RxJS.
  • O Observable recebe uma função que aceita um parâmetro subscriber_param. Este parâmetro é usado para emitir valores para os observadores inscritos.
Emissão de valores estatico:
  • subscriber_param.next(10): Emite o valor 10.
  • subscriber_param.next('Bruno'): Emite a string 'Bruno'.
  • subscriber_param.next(true): Emite o valor booleano true.
  • subscriber_param.next({{ '{' }} name: 'Bruno', age: 36 {{ '}' }}): Emite um objeto com as propriedades name e age.
  • subscriber_param.next({{ '[' }}1, 2, 3, 4{{ ']' }}): Emite um array de numeros.
  • subscriber_param.complete(): Indica que o Observable completou e não emitirá mais valores.
Emissão de valores Dinâmicos:
  • subscriber_param.next(values): Emite os valores dos formulários.
  • subscriber_param.complete(): Indica que o Observable completou e não emitirá mais valores.
  • Digite os valores:
    • Valores Submetidos:
    {{ submittedValues | json }}
Inscrição no Observable:
  • observable.subscribe(...): Inscreve-se no Observable para receber os valores emitidos.
  • O método subscribe aceita três funções de callback:
    • (res) => {{ '{' }} console.log(res); {{ '}' }}: Chamada sempre que um novo valor é emitido. O valor é registrado no console.
    • (err) => {{ '{' }} console.log(err); {{ '}' }}: Chamada se ocorrer um erro. O erro é registrado no console.
    • () => {{ '{' }} console.log('compleat the OBSERVABLE'); {{ '}' }}: Chamada quando o Observable completa. A mensagem 'compleat the OBSERVABLE' é registrada no console.
Subscription do Observable:
  • const subscription = observable.subscribe(...): Para se desinscrever declare o subscribe a uma variavel.
  • O método subscription deve ser chamado para se desinscrever do Observable:
    • subscription.unsubscribe(): Desinscreve-se do Observable.
      • Valores infinito com o unsubscribe defina os segundos para se desinscrever:
      {{ valor }}
